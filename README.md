#### 1. Project Setup
```bash
git clone https://github.com/galio-org/galio.git
cd galio
npm install or yarn install
```

#### 2. Project testing
Terminal cli:
`npm run ios` or `yarn run ios`

[Expo XDE client](https://docs.expo.io/versions/latest/introduction/xde-tour)
